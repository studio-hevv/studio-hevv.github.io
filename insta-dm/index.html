인스타 DM 분석
— 우리 둘 중 누가 더 진심일까?

인스타 DM을 올리면 노력·애정 두 점수로 알려드려요. 우리 둘 중 누가 더 좋아하는지, 데이터로 확인하세요.

폰으로 스캔

어떤 앱인가요

인스타그램에서 받은 DM 데이터를 올리면, 누가 먼저 말을 걸고 누가 더 반응하는지 분석해 노력·애정 두 점수로 보여줍니다. 관계 모드(썸·연인 / 친구·지인)를 고르면 상황에 맞게 해석해줘요.

사용 방법

  1. 인스타 DM 데이터 내려받기
    인스타그램 '내 정보 다운로드'에서 메시지 데이터를 요청해 받습니다.
  2. 파일 업로드
    토스 앱에서 인스타 DM 분석하기를 열고 받은 파일을 올립니다.
  3. 관계 진단 결과 확인
    상대를 고르고 관계 모드를 선택하면 노력·애정 점수와 해석이 나옵니다.

자주 묻는 질문

내 DM 데이터가 안전한가요?

분석에 필요한 값만 계산에 쓰며, 원본 대화 내용을 그대로 저장·공유하지 않습니다.

무료인가요?

요약 결과는 무료로 볼 수 있고, 점수 근거까지 담은 상세 리포트는 유료로 잠금 해제합니다.

어떤 데이터가 필요한가요?

인스타그램 '내 정보 다운로드'로 받은 메시지(DM) 데이터 파일이 필요합니다.

어떻게 실행하나요?

토스 앱에서 '인스타 DM 분석하기'를 열면 설치 없이 바로 사용할 수 있습니다.
