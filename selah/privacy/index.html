개인정보 처리방침

최종 업데이트: 2026년 6월 13일

본 개인정보 처리방침은 셀라(Selah) 예배 노트 앱(이하 "앱")을 사용하실 때 어떤 정보가 처리되는지 안내드립니다. 저희는 사용자의 프라이버시를 최우선으로 고려하며, 외부 서버에 데이터를 전송하거나 제3자와 공유하지 않습니다.

1. 수집·처리하는 정보

앱은 사용자가 직접 입력한 정보만 처리하며, 모든 데이터는 사용자 기기와 사용자 본인의 iCloud 계정에만 저장됩니다.

사용자가 직접 입력하는 정보

  • 예배 노트 본문, 설교자 이름, 말씀 주제, 성경 구절
  • 다짐·실천 항목
  • 예배 시간 및 딴짓(앱 이탈) 기록

자동으로 수집되는 정보 (로컬에만 저장)

  • 예배 모드 진입 시각, 종료 시각
  • 앱이 백그라운드로 전환된 시간 및 횟수 (집중도 통계용)

2. 정보의 저장 위치

항목저장 위치
예배 노트, 다짐 등 모든 콘텐츠기기 로컬 + 사용자의 iCloud 계정
위젯/Live Activity 표시용 데이터App Group (기기 로컬)
구독 정보Apple StoreKit (Apple 서버)

저희는 별도의 서버를 운영하지 않으며, 사용자 데이터를 수집하거나 전송받지 않습니다.

3. 정보의 사용 목적

  • 예배 노트 작성 및 열람
  • 다짐 실천 진행률 추적 및 위젯 표시
  • 예배 집중도 통계 계산 및 표시
  • Live Activity / Dynamic Island를 통한 진행 상태 표시
  • 구독 상태 확인 (Apple StoreKit 통해)

4. 정보의 제3자 제공

저희는 사용자의 어떠한 정보도 제3자에게 제공하지 않습니다. 분석 도구(Google Analytics, Firebase 등)나 광고 SDK도 사용하지 않습니다.

iCloud 동기화는 Apple의 서비스이며, 데이터는 사용자 본인의 iCloud 계정에만 저장됩니다. Apple의 개인정보 처리방침은 apple.com/kr/legal/privacy를 참고해 주세요.

5. 정보의 보관 기간 및 파기

  • 모든 데이터는 사용자가 직접 삭제하거나 앱을 삭제할 때까지 보관됩니다.
  • 개별 노트·실천 항목은 앱 내에서 언제든지 삭제 가능합니다.
  • 앱을 삭제하면 기기 내 데이터가 즉시 삭제됩니다. iCloud에 저장된 데이터는 iOS 설정 → Apple ID → iCloud → 저장공간 관리에서 삭제할 수 있습니다.

6. 사용자의 권리

  • 자신의 데이터 열람 (앱 내에서 직접 확인)
  • 데이터 수정 및 삭제 (앱 내 편집/삭제 기능)
  • 앱 삭제를 통한 모든 로컬 데이터 일괄 삭제
  • iCloud 동기화 비활성화 (iOS 설정에서 가능)

7. 알림 및 권한

  • 알림 (Notifications): 주일 예배 리마인더 (선택)
  • iCloud: 노트 동기화 (선택, iOS 설정에서 제어)
  • Live Activities: Dynamic Island/잠금화면 표시 (선택)

8. 14세 미만 아동의 개인정보

본 앱은 14세 미만 아동을 주 대상으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다.

9. 정책 변경

본 개인정보 처리방침이 변경될 경우 앱 내 안내 또는 본 페이지를 통해 사전 고지합니다.
