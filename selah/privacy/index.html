개인정보 처리방침

최종 업데이트: 2026년 6월 17일

본 개인정보 처리방침은 셀라(Selah) 예배 노트 앱(이하 "앱")을 사용하실 때 어떤 정보가 처리되는지 안내드립니다. 저희는 사용자의 프라이버시를 최우선으로 고려합니다.

1. 수집·처리하는 정보

사용자가 직접 입력하는 정보

  • 예배 노트 본문, 설교자 이름, 말씀 주제, 성경 구절
  • 다짐·실천 항목
  • 예배 시간 및 딴짓(앱 이탈) 기록

자동으로 수집되는 정보 (로컬에만 저장)

  • 예배 모드 진입 시각, 종료 시각
  • 앱이 백그라운드로 전환된 시간 및 횟수 (집중도 통계용)

서비스 개선을 위해 수집되는 정보 (제3자 서비스 경유)

  • 앱 사용 이벤트 (화면 조회, 기능 사용 횟수 등) — Firebase Analytics
  • 구독 상태 및 결제 이력 — RevenueCat
  • 광고 성과 측정을 위한 기기 식별자 (ATT 허용 시에만) — Meta · Google

2. 정보의 저장 위치

항목저장 위치
예배 노트, 다짐 등 모든 콘텐츠기기 로컬 + 사용자의 iCloud 계정
위젯/Live Activity 표시용 데이터App Group (기기 로컬)
구독 정보Apple StoreKit (Apple 서버)
앱 사용 이벤트Firebase (Google 서버, 익명화)
구독·결제 분석RevenueCat 서버

3. 정보의 사용 목적

  • 예배 노트 작성 및 열람
  • 다짐 실천 진행률 추적 및 위젯 표시
  • 예배 집중도 통계 계산 및 표시
  • Live Activity / Dynamic Island를 통한 진행 상태 표시
  • 구독 상태 확인 (Apple StoreKit · RevenueCat 통해)
  • 앱 기능 개선을 위한 사용 패턴 분석 (Firebase Analytics)
  • 광고 효과 측정 (ATT 허용 시에만, Meta · Google)

4. 제3자 서비스

앱은 서비스 운영을 위해 아래 제3자 서비스를 사용합니다. 각 서비스는 자체 개인정보 처리방침을 보유합니다.

서비스목적개인정보 처리방침
Apple iCloud 노트 동기화 apple.com
Apple StoreKit 구독 결제 apple.com
RevenueCat 구독 분석 revenuecat.com/privacy
Firebase Analytics (Google) 앱 사용 분석 policies.google.com/privacy
Meta (Facebook) 광고 성과 측정 (ATT 허용 시) facebook.com/privacy/policy
Google Ads 광고 성과 측정 (ATT 허용 시) policies.google.com/privacy

5. 광고 추적 (ATT)

앱은 Apple의 앱 추적 투명성(ATT) 정책에 따라, 광고 성과 측정을 위한 기기 식별자(IDFA) 접근 전 사용자의 명시적 동의를 구합니다.

  • 허용 시: IDFA를 통해 어떤 광고를 통해 앱을 설치했는지 측정합니다.
  • 거부 시: 광고 추적 없이 앱의 모든 기능을 동일하게 이용할 수 있습니다.

추적 설정은 iOS 설정 → 개인정보 보호 및 보안 → 추적에서 언제든지 변경할 수 있습니다.

6. 정보의 보관 기간 및 파기

  • 사용자가 직접 입력한 데이터는 삭제하거나 앱을 삭제할 때까지 보관됩니다.
  • 개별 노트·실천 항목은 앱 내에서 언제든지 삭제 가능합니다.
  • 앱을 삭제하면 기기 내 데이터가 즉시 삭제됩니다. iCloud 데이터는 iOS 설정 → Apple ID → iCloud → 저장공간 관리에서 삭제할 수 있습니다.
  • Firebase · RevenueCat 등 제3자 서비스에 전송된 데이터는 각 서비스의 보관 정책을 따릅니다.

7. 사용자의 권리

  • 자신의 데이터 열람 (앱 내에서 직접 확인)
  • 데이터 수정 및 삭제 (앱 내 편집/삭제 기능)
  • 앱 삭제를 통한 모든 로컬 데이터 일괄 삭제
  • iCloud 동기화 비활성화 (iOS 설정에서 가능)
  • 광고 추적 거부 (iOS 설정 → 개인정보 보호 및 보안 → 추적)

8. 앱 권한

  • 알림 (Notifications): 주일 예배 리마인더 및 매주 수요일 말씀 되새김 알림 (선택)
  • iCloud: 노트 동기화 (선택, iOS 설정에서 제어)
  • Live Activities: Dynamic Island/잠금화면 표시 (선택)
  • 앱 추적 (ATT): 광고 성과 측정 (선택, 거부 시에도 앱 전체 이용 가능)

9. 14세 미만 아동의 개인정보

본 앱은 14세 미만 아동을 주 대상으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다.

10. 정책 변경

본 개인정보 처리방침이 변경될 경우 앱 내 안내 또는 본 페이지를 통해 사전 고지합니다.
